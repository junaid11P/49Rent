# 49Rent

Production-ready camera rental experience built with Next.js 15, React 19, TypeScript, Tailwind CSS, Framer Motion, Supabase, and a premium minimalist UI.

## Features
- Premium landing experience with animated hero and glassmorphism
- Responsive product catalog and booking flow
- Customer and admin auth-ready routes
- Supabase-ready client/server helpers and validation schema
- SEO, sitemap, and robots support

## Getting started
1. Install dependencies
   npm install
2. Create a .env.local file with Supabase variables
3. Run development server
   npm run dev

## Environment variables
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=

## Supabase SQL schema
See the included SQL file in the repository root.

## Deployment
Deploy to Vercel and add the same environment variables in project settings.
